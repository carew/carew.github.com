How do tags work?

You can tag any document with the following syntax:

---
title: How to create a new page?
tags:
    - documentation
    - page
---

When a document is tagged, it's going to be added to the global variable carew.tags.<tag name>.

Moreover, few pages will be generated:

  • A first one which lists all tags.
  • A page per tag which lists all documents.

You can create a link to the first page with this syntax {{ path('tags') }} and to a specific tag: {{ path('tags/<tag name>') }}.

For the moment, tags pages are not customizable. Open an issue if you want to be able to customize them.
