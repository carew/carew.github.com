Installation#

You can use as many plugin as you want.

  1. Add the dependancy with composer.

  2. Register theme in the config.yml file:

    #config.yml
engine:
    extensions:
        - Carew\Plugin\Toc\TocExtension

That's it.

How to write a plugin#

The plugin must implements ExtensionInterface.

Carew will call ExtensionInterface::register and give itself to the extension. So the plugin can alter everything.

The plugin can access to:

  • The carew instance, of source
  • The container, an instance of pimple.
  • The event_dispatcher, an instance of Symfony EventDispatcher.

You can have a look to the CoreExtension for more information.
